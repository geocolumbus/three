Three.js is a 3D Javascript Library

  • HTML 5: Provides <canvas>, <svg>, CSS3D and WebGL renderers
  • Interact with 3D Objects in the Browser
  • Works with Chrome & Firefox - and supposedly IE9
  • To learn: Google "three.js" and read "WebGL Up and Running"
  • https://github.com/mrdoob/three.js/ - his examples
  • Google Sketchup: http://www.sketchup.com
  • My example: www.tallgeorge.com/3d/demo












Coding techniques

  • These are NOT sprites or flipbook 2D images - real browser objects.
  • Picking a 3D object - vector intersection between a point in front of the screen, passing through the plane that the mouse is on. First object intersected is selected.
  • Writing text on a 3D object: Use <canvas> element to convert text to image, paste image on object.
  • Animation (rotation) - Master animation loop changes position / rotation each "clock cycle"













Challenge

  • Not just for games - use 3D for navigation and information presentation.

  • With SVG element in HTML 5, adding 3D graphics is getting easier.
